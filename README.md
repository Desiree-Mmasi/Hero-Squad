# Hero Squad
#### This application enables users build their Squad teams and also build their own super-heroes.

#### Technologies Used
1. Intellij - This is the text editor that enabled us write Java Code
2. Gradle - Enabled us build tests
3. Git Terminal - This enabled us interact with the user and display their output
4. Heroku - Enabled the deployment of the project



## Setup/Installation Requirements
1. Download Intellij from https://www.jetbrains.com/idea/download/#section=linux
2. Visit my GitHub repository; User name Desiree-Mmasi and clone the following link:https://github.com/Desiree-Mmasi/Hero-Squad.git.You can also download the files into you folder and unzip it to access the files.
3. Open the project with Intellij.
4. Type gradle run in order to run the app from your local machine. 

## Behaviour Driven Development
| Behaviour                              | Input                            | Output                                                   |
|----------------------------------------|----------------------------------|----------------------------------------------------------|      
| The user lands on the  application     | Navigates to the desired instance| The user gets onto their desired instance in the website |
| The user chooses to form a squad       | Inputs Squad details             | The Squad formed is displayed on viewing formed squads   |
| The user chooses to form heroes        | Inputs the Hero details          | The Hero formed is displayed on viewing hero formed      |
